Quick Help - SMS Commands

Reply to any message with these commands:

  • HELP - Get instant help information including:
    • How to unsubscribe
    • Support contact information
    • Message frequency details
  • STOP - Immediately unsubscribe from all messages
    • Takes effect within seconds
    • You'll receive a confirmation message
    • No fees or penalties
  • UNSUBSCRIBE or CANCEL - Alternative opt-out commands

Email Support Guidelines

When contacting us via email, please include:

  • Your phone number (so we can locate your account)
  • A clear description of your question or issue
  • Any error messages you've received
  • Screenshots if applicable (for technical issues)
Privacy Note: We will never ask you for passwords, financial account information, or social security numbers via email. If you receive such a request, it is not from us.
