SMS Opt-In Consent Documentation

TCPA Compliance & Verification

Purpose of This Documentation

This page documents our SMS opt-in consent collection process to demonstrate full compliance with the Telephone Consumer Protection Act (TCPA), FCC regulations, and carrier requirements for toll-free messaging verification.

Compliance Statement: Our SMS consent process meets or exceeds all requirements for express written consent under TCPA regulations and carrier verification standards.

Express Written Consent Collection Method

We obtain valid express written consent through a comprehensive, multi-step verification process:

Step 1: Unchecked Consent Checkbox

Initial State: Checkbox is presented in an UNCHECKED state by default
Affirmative Action Required: User must manually click/tap to check the box
No Pre-Selection: The checkbox is NEVER pre-selected or checked by default
Validation: Form submission is blocked unless checkbox is actively selected

Step 2: Clear and Conspicuous Disclosure

The consent checkbox is accompanied by clear disclosure text that includes:

  • Service Description: What messages will be received (trading alerts, position updates)
  • Message Frequency: Approximate number of messages (10-30 per day)
  • Sender Identity: Messages from Raghuveer Ganta
  • Cost Disclosure: "Message and data rates may apply"
  • Opt-Out Instructions: "Reply STOP to opt-out"
  • Terms & Privacy Links: Direct links to full Terms and Privacy Policy

Step 3: Digital Signature via Form Submission

Phone Number Entry: User provides mobile number
Form Submission: Clicking submit button serves as digital signature
Consent Timestamp: Date and time of consent recorded
IP Address Logging: Optional IP address capture for verification

Step 4: Consent Record Retention

We maintain detailed records including:

  • Phone number that consented
  • Date and time of consent
  • Text of consent language presented
  • Evidence that checkbox was unchecked by default
  • User agent and source URL

TCPA Compliance Checklist

Our consent process satisfies all TCPA requirements:

✓ Prior express written consent obtained before ANY messages sent
✓ Checkbox is NOT pre-selected or pre-checked
✓ Clear disclosure of message purpose (trading alerts)
✓ Message frequency disclosed (10-30 msgs/day)
✓ Cost disclosure provided ("message and data rates may apply")
✓ Easy opt-out method disclosed (reply STOP)
✓ Terms of Service and Privacy Policy easily accessible
✓ Digital signature obtained via form submission
✓ Consent records retained for compliance verification
✓ Immediate STOP processing (opt-out within seconds)
✓ HELP command provides support information
✓ No marketing or promotional messages - alerts only
